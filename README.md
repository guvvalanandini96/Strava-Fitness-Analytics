# 🚴 Strava Fitness Data Analysis

## 📌 Project Overview

This project focuses on analyzing Strava fitness tracker data to understand user activity patterns, health metrics, and fitness behavior. Using SQL, Python, and Tableau, the project transforms raw fitness data into meaningful business insights through data cleaning, exploratory data analysis (EDA), and interactive visualizations.

---

## 🎯 Business Objective

The primary objective of this project is to analyze users' fitness activities and identify trends in physical activity, calorie expenditure, sleep behavior, and overall health. The insights generated can help improve fitness recommendations, user engagement, and wellness strategies.

## 📈 Key Insights

- Users with higher daily step counts generally burn more calories.
- Total distance traveled has a strong positive relationship with calorie expenditure.
- Most users spend more time in light physical activities than vigorous exercise.
- Sedentary behavior varies significantly among users.
- Daily activity patterns fluctuate across different days.
